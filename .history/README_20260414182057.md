# 📝 Task Scheduler App

## 📌 Objective

This web app allows users to:

* Create, view, edit, and delete tasks
* Schedule tasks
* View tasks on dashboard and calendar

---

## 🚀 Features

* Login with Email & Password
* Login using Email OTP
* Add / Edit / Delete tasks
* Mark tasks as completed
* View tasks in calendar
* Clean and simple UI

---

## 🛠️ Tech Stack

* Frontend:React
* Backend: Python  FastAPI
* Database: MongoDB
* API: REST API
* Version Control: Git
* Deployment: AWS

---

## 🔗 API (Sample)

* `POST /register`
* `POST /login`
* `POST /otp`
* `GET /tasks`
* `POST /tasks`
* `PUT /tasks/{id}`
* `DELETE /tasks/{id}`

---

## ⚙️ Setup

```bash
git clone <repo-link>
cd project
```

### Backend

```bash
pip install -r requirements.txt
python manage.py runserver
```

### Frontend

* Open `index.html`
